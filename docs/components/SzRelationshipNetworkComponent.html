Takes the data from a graph request and scans all entities found for relationship match keys. match keys found are broken in to constituant tokens and classified as either "DISCLOSED" or "DERIVED" types. The members of the return value for "DERIVED" or "DISCLOSED" are arrays of the entity ids found in the data that have that match key token present.
