bestEntityName
bestEntityName(entity: SzResolvedEntity)

A reusable function to obtain the best displayable name for an instance of SzResolvedEntity. This will first attempt to leverage the "bestName" and "entityName" fields if they exist on the specified parameter and are non-empty. Otherwise, it will do its best to find a name among the features of the entity. As a fall back a name is created using the entity ID.

Parameters :
Name Type Optional Description
entity SzResolvedEntity No

The SzResolvedEntity to get the name for.
Returns : string

This returns the best name for the specified entity, or "-" if the specified entity is null or undefined.
